digital library

an interactive virtual library containing every book i've read.

built with goodreads, open library api, and sentence transformers, an open-source embedding model framework. sentence embedding model all-MiniLM-L6-v2 was used to map book description and metadata to a 384-dimensional dense vector space of semantic clusters. TSNE (t-distributed stochastic neighbor embedding) was then applied to visualize the data in two dimensions.

coded in python, javascript, and html. you can find the offline vector encoding source code in colab and the frontend visualization code in github.

graph

a nearest-neighbors graph of 2-dimensional book embeddings. similar books are located nearer in space, forming visual clusters representing semantic similarity.
